# AZ State Remodeling LLC - Sitio Web Corporativo

## 📋 Descripción del Proyecto

Sitio web profesional para **AZ State Remodeling LLC**, una empresa de remodelación con licencia en Arizona. El sitio muestra los servicios de la empresa, permite a los clientes ver proyectos anteriores a través de un carrusel de imágenes, y cuenta con un sistema de reseñas con autenticación local.

## ✨ Características Principales

### 🏠 **Página Principal**
- Banner animado de "Under Remodeling"
- Hero section con llamado a la acción
- Sección de redes sociales integrada

### 🛠️ **Servicios Ofrecidos**
- **Flooring**: Carpeta, Tile, Vinyl
- **Painting**: Interior y Exterior
- **Drywall/Stucco Repairs**: Reparaciones y texturas
- **Bathroom Remodel**: Accesorios, puertas de vidrio, plomería
- **Kitchen Remodel**: Backsplash, gabinetes, encimeras
- **Custom Cabinets**: Diseño personalizado
- **Trim**: Baseboard, Crown Molding
- **Hardscape**: Concreto, sistemas de riego, pasto sintético

### 🖼️ **Galería de Proyectos**
- Carrusel interactivo de imágenes
- Organizado por servicio y subcategoría
- Navegación táctil para dispositivos móviles
- Modal con vista completa

### ⭐ **Sistema de Reseñas con Autenticación**
- **Login sin conexión externa** (solo localStorage)
- **Opciones de autenticación**:
  - 📧 Inicio de sesión con Email
  - 📞 Inicio de sesión con Teléfono
- **Validación de datos**:
  - Email válido (con @ y dominio)
  - Teléfono (mínimo 10 dígitos)
- **Persistencia de sesión** (se mantiene al recargar)
- **Avatares automáticos** con iniciales
- **Badge de "Verified"** en todas las reseñas
- **Calificación con estrellas** (incluye medias estrellas)
- **Carrusel de reseñas** con navegación
- **Formulario interactivo** para nuevas reseñas

### 📱 **Diseño Responsive**
- Optimizado para móviles, tablets y desktop
- Menú hamburguesa en dispositivos móviles
- Carruseles adaptables
- Botones flotantes de contacto

### 📞 **Contacto**
- Formulario de contacto funcional (FormSubmit.co)
- Botones flotantes: WhatsApp, Llamada, SMS
- Información de contacto y redes sociales

## 🛠️ **Tecnologías Utilizadas**

- **HTML5** - Estructura semántica
- **CSS3** - Estilos personalizados con diseño responsive
- **JavaScript** - Funcionalidad interactiva
- **Font Awesome 6** - Iconografía
- **Google Fonts** - Tipografía Poppins
